# Exercício 01 - Função ou Procedimento

## Str()

É uma **função**, pois recebe um número e retorna esse valor convertido para texto.

## QOut()

É um **procedimento**, pois apenas exibe uma informação na tela.

## Date()

É uma **função**, pois retorna a data atual do sistema.

## Len()

É uma **função**, pois retorna o tamanho de uma string ou de um array.

## Upper()

É uma **função**, pois retorna uma string convertida para letras maiúsculas.

## ClearScreen()

É um **procedimento**, pois apenas limpa a tela.
