# Exercício 1 — Isto é um algoritmo válido?

Um algoritmo precisa ter 3 características:

- **Finito** → precisa terminar
- **Ordenado** → os passos devem estar em sequência
- **Sem ambiguidade** → qualquer pessoa deve interpretar da mesma forma

---

## a) "Some dois números."

❌ **Não é um algoritmo válido.**

**Por quê?**

Porque está incompleto. Ele diz apenas para somar, mas não explica:

- de onde vêm os números
- onde mostrar o resultado

Um algoritmo correto seria:
- Leia o primeiro número
- Leia o segundo número
- Some os dois números
- Mostre o resultado


---

## b) "Leia um número. Se for positivo, mostre 'positivo'. Senão, mostre 'negativo'."

✅ **É um algoritmo válido.**

**Por quê?** Possui:

- ✔️ Entrada de dados (Leia)
- ✔️ Decisão (Se / Senão)
- ✔️ Saída (Mostre)
- ✔️ Termina após mostrar o resultado

---

## c) "Fique repetindo até acabar."

❌ **Não é válido.**

**Por quê?** O algoritmo nunca informa:

- Acabar o quê?
- O que repetir?

Pode repetir para sempre → é **infinito** e **ambíguo**.

---

## d) "Pegue um ingrediente qualquer e cozinhe."

❌ **Não é válido.**

**Por quê?** Existem muitas interpretações:

- Arroz? Carne? Ovo? Macarrão?

Também não informa:

- Quanto tempo
- Temperatura
- Modo de preparo

É um algoritmo **ambíguo**.
